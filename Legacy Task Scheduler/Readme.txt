Adding Scheduled Tasks for Add2Exchange permissions on Server 2008 or 2008R2:

1. Use the GUI to update any credentials

2. Task Scheduler should be open. On the right side under Actions, click on Import Task.

3. The foler "Legacy Task Scheduler" should be open. Choose wich .xml file you would like to import from here. 

Note* For Dual permissions, import both on-premise/O365 dual .xml files for either distribution groups or giving permissions to all.
